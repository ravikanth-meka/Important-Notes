To enable swagger UI Setp 1: Add following in POM.xml io.springfox springfox-swagger2 2.4.0 io.springfox springfox-swagger-ui 2.4.0 Step 2: Add Spring configuration @Configuration @EnableSwagger2 public class SwaggerConfig { private static final Set DEFAULT_PRODUCES_AND_CONSUMES = new HashSet(Arrays.asList("application/json", "application/xml")); @Bean public Docket api(){ return new Docket(DocumentationType.SWAGGER_2) .produces(DEFAULT_PRODUCES_AND_CONSUMES) .consumes(DEFAULT_PRODUCES_AND_CONSUMES); } } To Access Swagger UI: http://localhost://swagger-ui.html#!
